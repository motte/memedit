# memedit
Playing around with iOS code architecture (explicit MVVM) with TikTok-esque Reddit feed.

## Some Features
1. List of Hot posts.
2. View different subreddits.
3. Share posts.
4. View likes count.
5. View comments count.
6. Dumb light/dark modes.
